==================================================
INSTRUCCIONES PARA CREAR EL LAUNCHER PARA WINDOWS
==================================================

Introducir la carpeta lib y la carpeta bin del jre al directorio "jre" de esta carpeta.
Mover el jar a esta carpeta.
Abrir el archivo wrapperConfig.xml con http://launch4j.sourceforge.net/ .
Retocar algún aspecto de la configuración si fuese necesario.
Clicar en el botón del engranaje para crear el launcher.
